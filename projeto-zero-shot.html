Categorização Automática de Produtos

Zero-Shot Classification (NLI) para limpar descrições de itens e sugerir categorias padronizadas.

Zero-shot NLI Hugging Face

Problema

Coluna “itens” de pedidos/produtos chega suja e despadronizada. Cadastros manuais geram erros e dificultam o agrupamento.

Objetivo: organizar automaticamente as descrições e atribuir categorias coerentes.

Público-alvo

  • Gestores de catálogo e marketplace
  • Equipes de dados e governança
  • Times de logística e compras

Abordagem

  • Zero-Shot Classification (NLI) usando Hugging Face
  • Modelos BERT/DeBERTa/BART para inferir categorias sem dataset anotado
  • Regras simples para expandir abreviações e gerar uma descrição legível

Dataset & Modelos

  • Arquivos locais: pedidos.csv e produtos.csv (TP1)
  • Pipeline zero-shot-classification com modelo NLI (ex.: facebook/bart-large-mnli)

Funcionalidades

  • Receber descrição suja e retornar categoria padronizada
  • Gerar uma descrição mais legível a partir de abreviações e siglas
  • Mostrar top 3 rótulos candidatos com scores
  • Demo 100% estática, usando demo_results.json pré-gerado

Demo (offline)

Use uma descrição suja e clique em "Classificar". Os resultados vêm de public/demo_results.json gerado previamente pelo script demo_zero_shot.py.
